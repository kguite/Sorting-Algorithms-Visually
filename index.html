Pixel Sorting Algorithms

Welcome! Trying to understand sorting algorithms?

This is a site that contains resources and visualizations for the five most commonly used: Bubble, Merge, Quick, Insertion and Selection.

Notes:

  • Time complexity is not accurate, this is just a visualization.
  • Merge sort creates a stack to be used, so it may run a bit slow at first.
  • For interesting visual effects, change the 'pixelDistance' variable in the JavaScript panel. This variable is used to set the distance between each pixel value in the image data array. The correct value is 4 since there are 4 pixels, RGBA.
  • For faster runtimes, make the 'imageSize' variable smaller
